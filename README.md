# FoodRescue
A platform where the volunteers, Restraunts and NGOs come together to make sure noone sleeps hungry. Resources are always available it's just some effort which is required for channelizing resources to right person to make sure every needful person receives what is necessary for live.

Technology Used:
HTML/CSS/JS
React.js (v19)
Vite (Bundler/Build Tool)
Typescript for Type checking
ESLint
Bootstrap
